# 💰 Interest Calculator (Python Project)

## 📌 Overview

The **Interest Calculator** is a simple yet practical Python project that calculates **Simple Interest** and **Compound Interest** based on user input.
It is designed to help users quickly compute interest values and understand financial calculations.

This project demonstrates the use of Python for mathematical computations and basic financial analysis.

## 🧮 Formulas Used

### Simple Interest

SI = (P × R × T) / 100

### Compound Interest

CI = P × (1 + R/100)^T - P

Where:

* P = Principal
* R = Rate of Interest
* T = Time
